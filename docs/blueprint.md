# **App Name**: Remotized IT Solutions

## Core Features:

- Gooey Desktop Navigation: Interactive 'gooey' style navigation menu for desktop users, enhancing UX with modern design.
- Prism Hero Animation: Rotating 3D prism animation in the hero section to capture attention and showcase modern tech focus.
- Animated Content Sections: Sections animate into view upon scrolling using GSAP's ScrollTrigger for an engaging experience.
- Portfolio Carousel: Showcase previous projects using the shadcn/ui carousel component.
- Testimonial Carousel: Display client testimonials in a carousel format, demonstrating trust and satisfaction.
- Fluid Cursor with Trail: Site-wide fluid cursor effect that leaves a colorful trail, adding a unique visual element.
- Responsive Layout: Fully responsive design ensures optimal viewing experience across all devices, including mobile hamburger menu.

## Style Guidelines:

- Primary color: Vibrant cyan (#00FFFF) to represent innovation and technology.
- Background color: Very dark desaturated blue (#0A0F14) to ensure readability and a modern feel.
- Accent color: Light cyan (#88FFF2) to highlight key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern look. Note: currently only Google Fonts are supported.
- Use clean, minimalist icons from lucide-react throughout the site to represent services and features.
- Single-page layout with smooth scrolling to all major sections for intuitive navigation.
- Implement GSAP ScrollTrigger for animating content sections, ensuring elements fade in and slide up on scroll.
